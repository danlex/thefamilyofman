# Data

Structured catalogs of the exhibition.

| File | Contents |
|---|---|
| `photographs.csv` | the 503 photographs of the exhibition |
| `photographers.csv` | the 273 contributing photographers |
| `sections.csv` | Steichen's thematic sections |

All files are CSV with column headers.
